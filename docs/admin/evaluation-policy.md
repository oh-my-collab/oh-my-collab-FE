# 평가 참고 정책

## 1. 목적
- 본 시스템은 인사평가 **참고자료**를 제공하기 위한 운영 도구다.
- 시스템이 최종 인사등급을 자동 확정하지 않는다.
- 최종 평가는 관리자/리더의 검토와 조직 정책에 따라 수동 확정한다.

## 2. 평가 기준(정량 + 정성)
- 정량(주기 기준):
  - 실행: 완료 작업 난이도/완료량
  - 문서: 문서 업데이트 횟수
  - 목표: KR 업데이트 활동
  - 협업: 댓글/리뷰/블로커 해소 이벤트
- 정성:
  - 관리자 메모
  - 문서/리뷰 내용의 맥락
  - 팀 기여도(지식 공유, 협업 품질)

## 3. 가중치 운영 원칙
- 주기마다 가중치를 설정한다.
- 가중치 합은 100이어야 한다.
- 가중치 변경은 기록되며(`performance_cycle_updated`) 변경 사유를 남기는 것을 권장한다.

## 4. 공정성/편향 방지 원칙
- 단일 지표로 판단하지 않는다.
- 업무 성격 차이(직무/역할/스코프)를 함께 고려한다.
- 리뷰 확정 전, 최소 1회 이상 교차 검토를 권장한다.
- 개인 민감정보나 평가와 무관한 요소를 근거로 사용하지 않는다.

## 5. 리뷰 프로세스
1. 초안 작성
  - 증거팩 확인
  - 관리자 메모/평점 초안 입력
2. 검토
  - 팀 리더 또는 owner와 기준 일치 여부 확인
3. 확정
  - `확정 및 잠금`으로 리뷰 종료
  - 필요한 경우 CSV/PDF 내보내기로 보관

## 6. 보존 및 접근
- 기본 보존 기간: 12개월
- 접근 권한:
  - `owner`: 전체 관리자 기능 + 관리자 지정/해제
  - `admin`: 주기/리뷰/내보내기
  - `member`: 관리자 영역 접근 불가
